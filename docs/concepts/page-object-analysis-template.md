# Page Object Analysis Template

## Page Purpose: [Name]
**User Goal**: View, Filter and sort different products

### Interactive Elements
- Title of product (link)
- Add to card/Remove button
- Cart icon counter
- Burger menu (All Items, About, Logout, Recent App state)
- Footer social links  

### Display Elements  
Header:
- Cart icon + Cart counter
- Burger menu
- Title of the page (Swag Labs)
- Sub-title of the page (Products)
Body:
- A two-column grid with 2 products each row
Each product containing:
    - Name of a product
    - Short Description of a product
    - Price of a product
    - Image of a product
- A filter with 4 drop-down options (Name A-Z, Name Z-A, Price High to Low, Price Low to High)
Footer:
- Copyright text
- Social links (twitter, LinkedIn, Facebook)


### User Actions (Methods)
- Clicking on links (socials) - Navigates to Saucelabs' social media pages
- Clicking on a product title link - Opens up product details page
- Clicking on a product image - Opens up product details page
- Clicking on the cart icon - Opens up the cart page
- Clicking on the burger menu - Opens up the menu bar
- Clicking on the filter - Opens up the filter drop-down menu
- Selecting a filter option - Filters the products based on the selected option
- Clicking on the "Add to cart" button - Adds the product to the cart
- Clicking on the "Remove" button - Removes the product from the cart

### Edge Cases to Handle
- I don't see any edge cases for this page.
